Comunicação na Internet

Exploração interativa dos protocolos de rede

Camada 7

HTTP / HTTPS

A linguagem universal da web para transferência de dados entre cliente e servidor.

Camada 7

DNS

O tradutor da rede: transforma nomes de domínios amigáveis em endereços IP legíveis por máquinas.

Tempo Real

WebSockets

Conexões persistentes e bidirecionais, permitindo fluxo de dados contínuo sem reabrir conexões.

TCP / IP

Base da Comunicação

A suíte de protocolos que governa como os dados são empacotados, endereçados e transmitidos. O TCP garante a entrega, enquanto o IP cuida da rota.

Arquivos

FTP / SFTP

Protocolos robustos focados exclusivamente no envio e recebimento de grandes volumes de arquivos.
